# Aurix – Digital Jewellery Marketplace (MVP)

Aurix is a mobile-first digital jewellery marketplace designed to connect customers with verified jewellers through a secure and modern platform.  
This project focuses on improving trust, accessibility, and efficiency in the jewellery industry using digital technology.

This repository contains the **Minimum Viable Product (MVP)** developed as part of an academic project.

---

## Project Overview

Aurix provides a centralized platform where:

- Customers can browse jewellery and communicate with jewellers
- Jewellers can manage their products and verification
- Administrators can monitor and control platform activities
- AI-assisted jewellery design is available at a basic level

---

## MVP Features

- User authentication with role-based access
- Jeweller verification and product management
- Customer browsing and chat functionality
- AI jewellery design generator (basic integration)
- Admin approval and monitoring system
