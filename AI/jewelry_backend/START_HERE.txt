🆘 COMMON ISSUES & FIXES
════════════════════════════════════════════════════════════════════════════════
"Backend Offline ❌"
  → Check: python3 app.py is running
  → Fix: Start the server in terminal

"Model is loading..."
  → Normal: First request takes 30-60 seconds
  → Fix: Wait and try again

"Connection refused"
  → Check: Wrong URL in jewelry_ai_service.dart
  → Fix: Use correct IP/URL for your device

"Image not displaying"
  → Check: AndroidManifest.xml has internet permission
  → Check: Backend URL is correct
  → Fix: Verify permissions and URL

"Port 5000 in use"
  → Check: AirPlay Receiver is on (Mac)
  → Fix: Disable in System Settings → General → AirPlay
